Description:
------------

vBulletin addon. Allows user to upload large pictures and crop avatar/profilepic from it. User should select his face on uploaded file.

Special notes:
--------------
1. GD shall be installed on your system

2. In order to convert existing profile pics to big ones please follow this procedure.
   a.PLace file from tools directory into the site root
   b.launch the file
